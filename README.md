# Projeto: Banco 360 - IA para Personalização e Eficiência no Setor Bancário

## 📒 Descrição
O projeto "Banco 360" utiliza tecnologias de IA generativa para transformar a experiência bancária, criando soluções personalizadas para clientes e otimizando processos internos. O foco é oferecer um atendimento mais humano, ágil e eficiente, além de explorar oportunidades de inovação no relacionamento com os clientes e na gestão bancária.

## 🤖 Tecnologias Utilizadas
ChatGPT: Para criação de assistentes virtuais e roteiros de conteúdo.
FlexChip: Para criação do vídeo

## 🧐 Processo de Criação
Identificação do Problema:
Mapeamento de dificuldades frequentes dos clientes (ex.: demora no atendimento, falta de orientação financeira).
Levantamento de processos internos que podem ser otimizados.

## 🚀 Resultados
Desenvolvimento das práticas utilizando as ferramentas de IA estudadas no curso.

## 💭 Reflexão (Opcional)
A IA não é apenas uma ferramenta de automação, mas um caminho para fortalecer o relacionamento entre bancos e clientes, trazendo mais empatia, eficiência e personalização ao setor bancário.
